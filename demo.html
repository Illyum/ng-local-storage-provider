Illyum Angular Local Storage

Demo

Illyum Angular Provider Local Storage provee un mecanismo para tener acceso al almacenamiento local del navegador con las siguientes características funcionales:

  • Permite almacenar elementos e identificarlos por medio de una llave (key). Los elementos pueden ser cadenas de texto, valores enteros o punto flotante, arreglos y objetos.
  • Permite recuperar el valor de un elemento por medio de su llave (key).
  • Permite borrar un elemento por medio de su llave (key).
  • Permite recuperar las llaves (keys) de los elementos almacenados en el Local Storage.
  • {{key}}

Uso

Dependencias:
  • Angular JS http://angularjs.org/
  • Illyum Angular Local Storage Moduleillyum-ng-local-storage-provider.js
Ejemplo Java Script
  var controller = function(illyumLocalStorage) {

  // Para guardar un elemento en el Local Storage
  illyumLocalStorage.putItem('the key goes here', 'the value goes here');

  // Para recuperar un elemento almacenado en el Local Storage
  var value = illyumLocalStorage.getItem('the key goes here');

  // Para borrar un elemento del Local Storage
  illyumLocalStorage.deleteItem('the key goes here');

  // Para obtener la lista de las llaves almacenadas en el Local Storage
  var keysInTheLocalStorage = illyumLocalStorage.getKeys();
  
}

Guía del programador

Método Argumentos Descripción Valor de retorno
putItem key, value Guarda un elemento (key-value o llave-valor) en el Local Storage Ninguno
getItem key (llave) Recupera un elemento almacenado en el Local Storage El valor almacenado
deleteItem key (llave) Borra un elemento key-value (o llave-valor) del Local Storage Ninguno
getKeys Ninguno Recupera las llaves de los elementos almacenados en el Local Storage Un arreglo Java Script
